# 🚀 Udada (우다다)
<img src="https://udada.vercel.app/udada-logo.png">
> 부경대학교 최신 공지사항을 이메일로!

## 📌 프로젝트 소개 (About Project)
키워드 설정을 통해 학교 홈페이지의 공지사항을 이메일로 쉽게 받아볼 수 있습니다.
* **배포 링크**: [서비스 바로가기](https://udada.vercel.app) 

### ✨ 주요 기능 (Key Features)
* **기능 1**: 카카오톡 로그인
* **기능 2**: 학교 공지사항 크롤링
* **기능 3**: 설정된 키워드의 공지사항을 이메일로 전송
